# Vista Calendar

**Vista Calendar** is a personal finance app that lets you forcast and track your buduget and expenses from a calendar view.

## Highlights

- **Local-first desktop experience** — all desktop data lives in local SQLite. Additional encryption features coming soon. 
- **Backup & restore** — export, import, and restore with validation. 

## Tech Stack

- **Electron** (v41) — native desktop shell
- **React** (v19) + **React Router** — UI
- **Vite** — build tooling
- **SQLite** (better-sqlite3) — local storage
- **Tailwind CSS** + shadcn/ui — styling

## Platform

Built and tested for **macOS** (Apple Silicon / arm64). macOS 26+ Tahoe is
required for the native glass effect.
